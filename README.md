# COVID-19 Global Analysis Dashboard (Jan–Jul 2020)

## 📌 Overview
This project analyzes global COVID-19 data from January to July 2020 using **Excel dashboards**.  
It highlights key pandemic trends such as confirmed cases, recoveries, deaths, active cases, growth rates, and global spread.

The dashboard is designed to be **business-oriented**, with KPI cards, sparklines, slicers, and charts that provide actionable insights.

---

## 🎯 Features
- **KPI Cards**: Headline metrics (Confirmed, Recovered, Deaths, Active) with sparklines for trend context.
- **Interactive Slicer**: Filter by month (Jan–Jul).
- **Trend Charts**:
  - Confirmed vs Recovered vs Deaths vs Active Cases
  - Global Expansion (Number of countries affected)
  - Active Case Trend (Healthcare pressure)
  - Monthly New Cases, Deaths, Recoveries
  - Mortality vs Recovery Ratios
  - Growth Rate Comparison
- **Business Questions Answered**:
  - Which days had new cases above the 90th percentile?
  - Correlation between new cases and new deaths.
  - Month-over-month growth in confirmed cases.
  - Recovery and mortality rate trends.
  - Quarter with highest active case burden.

---

## 🛠 Tools & Techniques
- **Excel**:
  - PivotTables & PivotCharts
  - Sparklines
  - Conditional Formatting
  - Forecasting (TREND, FORECAST.LINEAR)
  - Custom Number Formatting (Millions/Billions)
- **Data Cleaning**:
  - Day-wise raw data transformed into monthly summaries
  - Calculated metrics (growth %, recovery %, mortality %)

---

## 📊 Insights
- Cases surged rapidly after March, peaking in July.
- Spread plateaued at ~187 countries by April.
- Recovery ratios improved steadily, mortality declined.
- Growth rate dropped sharply after January, then declined gradually.
- Strong correlation (r ≈ 0.72) between new cases and new deaths.

---

## 🚀 How to Use
1. Download the Excel file from this repository.
2. Open the **Dashboard** sheet to explore interactive visuals.
3. Use the **Date slicer** to filter by month.
4. Review KPI cards and charts for insights.
